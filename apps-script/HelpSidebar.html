❓ Help

1 Prepare your sheet

Make sure your "Orders" sheet has columns: Order ID, Customer, Phone, Address, Wilaya, Commune, Product, Quantity, Price, Delivery, Provider, Status, Tracking, Notes, Date.

2 Set up API keys

Go to Settings → enter your provider API keys as JSON.

3 Send orders

Select "Send Orders" from the Delivery Tool menu. Only rows with empty or "Pending" status are processed.

4 Track statuses

Use "Update Statuses" to refresh delivery statuses from providers.

Supported Providers

  • Yalidine, ZR Express, DHD, Allo, 48HR
  • WorldExpress, Anderson, Guepex, Samex
  • Ovred, Navex, Elogistia, Colireli, Assil
  • Areex, Ecom, Noest, Salva, Expedia
